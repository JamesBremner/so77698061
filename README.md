# Email Status

A windows application to display email status

## Input

A CSV file saved from excel like this:

```
   "title1", "Amazon", "X"
   "title1", "Apple", "X", "sent"
   "title1", "Meta", "X", "sent"
   "title1", "Microsoft", "X", "sent"
```
